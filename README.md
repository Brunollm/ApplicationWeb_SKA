# Lista de Tarefas

> Status: Concluido ✅

### Essa é uma ApplicationWeb feita para gerenciar tarefas onde pode-se criar, atualizar e remover uma ou mais tarefas, você também colocar descrição, prioridade e status.

## Os campos para preenchimento são:

+ Nome
+ Descrição da Tarefa
+ Prioridade
+ Status da Tarefa

## Tecnologias Usadas:

+ C#
+ ASP.NET
+ HTML
+ CSS
+ EntityFramework
+ SQL Server
